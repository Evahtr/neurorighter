Neurorighter

Closed Loop, Open Source Electrophysiology

Download this project as a .zip file Download this project as a tar.gz file

==About== NeuroRighter is an open-source electrophysiology platform for conducting closed-loop, multichannel neural recording and stimulation experiments. It was originally produced by [https://neurolab.gatech.edu/labs/potter the Potter Lab] at Georgia Tech, and has enabled [http://pwp.gatech.edu/optoclamp/ pioneering neuroscience research] through precise electrical and optical control of neural tissue.

  • Extensive documentation can be found on the [https://sites.google.com/site/neurorighter/ NeuroRighter home page].

  • The NeuroRighter API reference can be found [https://potterlab.gatech.edu/main/neurorighter-api-ref/ here].

  • A bug does not exist until it is submitted to the [http://code.google.com/p/neurorighter/issues/list Issues Page].

  • The [https://groups.google.com/forum/?fromgroups#!forum/neurorighter-users NeuroRighter User's forum] is a great place to get information on constructing/using NeuroRighter.

  • Code statistics on the NeuroRighter repository can be found [https://www.ohloh.net/p/neurorighter here].

==Installation== To install NeuroRighter, go to the [http://code.google.com/p/neurorighter/downloads/list Downloads Page] and get the latest zip archive. Unzip the folder and run Setup.exe. Follow the instructions presented on your computer monitor.
